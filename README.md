# Shorter

A simple URL shortener written in Go using Fiber, Templ & TailwindCSS

## Usage

Install Dependencies

```sh
pnpm i
```

Develop

```sh
pnpm dev:css      # run TailwindCSS watcher
pnpm dev:template # run Templ compiler
pnpm run          # run Fiber server
```

Build

```sh
pnpm build
```

Run

```sh
pnpm start
```
